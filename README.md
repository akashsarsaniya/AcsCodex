# AK Service - Digital Marketing Agency

A modern React application for AK Service, a digital marketing agency website inspired by Thrive Agency.

## Features

- **Home Page**: Showcases services, results, testimonials, and call-to-action sections
- **About Page**: Information about the company, team, and why choose us
- **Contact Page**: Contact form and company information
- **Responsive Design**: Mobile-friendly layout
- **Modern UI**: Clean and professional design

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```
